ONS On Demand

A SasS solution to allow instituions to purchase courses in a new way.

Problem

We completed the project as a way for company admins to be able to enroll and track their staff in courses. It also allowed them to pull comprehensive reports. Prior to this admins could purchase vouchers but there was no tracking on redemption or course completion.

Solution

A new product built in-house by software engineers and myself. A separate API product was built for insitutions that would want to build out their own version of the product.

This allowed large institutions to purchase courses in bulk via a licensing management system. instituions such as Mickesson Healthcare Corporation can now track how their nurses are learning and assign and unassign licenses via this SasS tool.
